## Available Schemes

### Classic
The default theme with a professional and clean appearance.
```css
/* Light */
--theme-primary: #007AFF;
--theme-secondary: #5856D6;
/* ... */

/* Dark */
--theme-primary: #0A84FF;
--theme-secondary: #5E5CE6;
/* ... */
```

### Ocean
Cool blues and teals for a calming experience.
```css
/* Light */

--theme-secondary: #0077B6;
/* ... */

/* Dark */

/* ... */
```
